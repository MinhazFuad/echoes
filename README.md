# Project Name : Echoes

## Overview
A silent, emotion‑driven visual platform where users express and explore meaning without text.

## Problem
Language limits emotional expression and interpretation.

## Solution
A non‑verbal system using colors, shapes, images, and interaction.

## User Flow
Landing → Emotion → Explore → React → Personalize → Create → Discover

## Core Features
- Emotion‑based discovery
- Silent posting system
- Non‑verbal reactions
- Private & saved content
- Personalized feed

## Tech Stack
- Next.js
- Prisma
- Database (PostgreSQL / SQLite)
- Auth system
- Cloud image storage
